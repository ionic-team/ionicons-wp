Add over _900_ free and open source mobile, iOS, Android, and social media icons to your wordpress site with the only official [Ionicons](http://ionicons.com/) WordPress plugin.

== Description ==

This plugin adds a simple `[icon]` shortcode for adding icons. For example, add the wonderful earth icon with `[icon name=ion-earth]`. You can
also use this plugin with HTML like this: `<i class="ion-earth"></i>`

== Installation ==

1. Unzip the download package
2. Upload `ionicons` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
